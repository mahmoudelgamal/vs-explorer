# Published Link

https://vs-explorer.netlify.app/

## How to setup

1. clone the repo

```shell
git clone git@github.com:mahmoudelgamal/vs-explorer.git
```

2. install dependencies

```shell
yarn
```

## How to start

```shell
yarn dev
```

## How to test build locally

1. run the build

```shell
yarn build
```

2. run the unit tests

```shell
 yarn test:cov
```
